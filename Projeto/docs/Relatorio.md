Dashboard para otimizar a gestão de tarefas do dia a dia.

---

A IA foi aplicada para desenvolver uma interface responsiva em HTML, CSS e JavaScript puro, integrada a um dashboard com estatísticas visuais. A aplicação permite criar, editar, deletar e concluir tarefas, além de gerenciá-las por status e prioridade através de um gráfico simples de produtividade. Ademais, é possível estipular horários ao longo do dia, marcar os dias concluídos e acompanhar o percentual de progresso até a data final de conclusão.

Na unidade 03, iremos aprender a combinar velocidade com segurança, funcionalidade com design e ambição com viabilidade.

**Objetivos:**

Preparar o estudante para ser um desenvolvedor ágil e versátil.

Ensinar discernimento no uso de ferramentas low code e no code.

Aprender a criar, testar e escalar aplicações inteligentes.

Estudar o uso de ferramentas específicas.

Capacitar a elaborar protótipos.
